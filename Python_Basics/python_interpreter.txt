
To open the Python Interpreter: 
type python3 (for python3)
     python3

You should see a prompt like:
Python 3.x.x (default, date, time) [GCC version] on linux
 Type "help", "copyright", "credits" or "license" for more information.
 >>>

Example of Adding Two Numbers in the Python Interpreter

1. Add Two Numbers:
   - Type:
     >>> 1 + 2

2. View the Result:
   - Press Enter, and the output will be:
     3

3. Exit the Interpreter:
   - Type:
     >>> exit()
   - Or use Ctrl + D (Linux/macOS) or Ctrl + Z then Enter (Windows).
